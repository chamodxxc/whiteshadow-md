HOW TO CONNECT

Step 1: Get pair code

Choose a Server from the options above and Enter your phone number to get the Pairing Code.

Step 2: Whatsapp setup

Copy pair code and Go to WhatsApp > Linked Devices > Link with Phone Number.

Step 3: connect to whatsapp

Enter the code and wait for the bot to initialize.Type .alive to check if the bot is active.
